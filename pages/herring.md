    <h1 id="bulma-clean-theme-demo-website">Bulma Clean Theme demo website</h1>

<p>This website showcases the options for the Bulma Clean theme. The theme is available as a ruby gem or can be used with GitHub pages.</p>

<p><a href="https://badge.fury.io/rb/bulma-clean-theme"><img src="https://badge.fury.io/rb/bulma-clean-theme.svg" alt="Gem Version" /></a>
<img src="https://img.shields.io/gem/dt/bulma-clean-theme.svg" alt="Gem" /></p>

<h2 id="ruby-gem">Ruby Gem</h2>

<p>The ruby gem is available on the Ruby Gems website at the following location. <a href="https://rubygems.org/gems/bulma-clean-theme">https://rubygems.org/gems/bulma-clean-theme</a></p>

<h2 id="github-pages">GitHub Pages</h2>

<p>The theme can be used with GitHub Pages bu setting the <code class="language-plaintext highlighter-rouge">remote_theme</code> in your Jekyll sites <code class="language-plaintext highlighter-rouge">_config.yml</code></p>

<div class="language-yml highlighter-rouge"><div class="highlight"><pre class="highlight"><code><span class="na">remote_theme</span><span class="pi">:</span> <span class="s">chrisrhymes/bulma-clean-theme</span>
</code></pre></div></div>

<h2 id="instructions">Instructions</h2>

<p>For full instructions, please see the Readme at the GitHub repo:
<a href="https://github.com/chrisrhymes/bulma-clean-theme/blob/master/README.md">https://github.com/chrisrhymes/bulma-clean-theme/blob/master/README.md</a></p>

<h2 id="page-layouts">Page Layouts</h2>

<p>This demo site showcases the available page layout options.</p>

<ul>
  <li>Page With Sidebar</li>
  <li>Page Without Sidebar</li>
  <li>Page With Menubar</li>
  <li>Page With Tabs</li>
  <li>Page Without Footer</li>
  <li>Page Without Hero</li>
  <li>Page With Contents</li>
  <li>Landing Page With Callouts</li>
  <li>Sponsors Page</li>
  <li>Image Gallery</li>
  <li>Blog</li>
  <li>Post</li>
</ul>
